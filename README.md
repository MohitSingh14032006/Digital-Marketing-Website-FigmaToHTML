# Responsive Website

A modern and responsive website developed from a Figma design with a strong focus on clean UI, smooth user experience, and responsive layouts across all devices.

##  Features

- Fully responsive design for mobile, tablet, and desktop
- Clean and minimal user interface
- Pixel-perfect implementation from Figma
- Smooth layout and organized content structure
- Modern typography and spacing
- Reusable and maintainable code structure
- Fast and user-friendly experience

## 🛠️ Technologies Used

- HTML5
- CSS3

## 🎨 Design Approach

The website was built by carefully converting the Figma design into a fully functional and responsive web interface. Special attention was given to:

- Consistent spacing and alignment
- Responsive grids and flexible layouts
- Neat color combinations and typography
- Better readability and accessibility
- Optimized UI for different screen sizes

## 📱 Responsiveness

The website is optimized for:

- Mobile Devices
- Tablets
- Laptops

## 🚀 Project Goal

The main objective of this project was to create a visually appealing and responsive website while maintaining a clean and professional user interface.
